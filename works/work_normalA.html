    Problem & Solution

    • "grid 안 컨텐츠 제어"
    • 안에 이미지가 세로로 넘침
    • 이미지 사이즈 조절을 가로가 아닌 세로로 함.
    • grid 안 화면 넓이에 맞는 정사각형의 값을 주고 싶은데 높이는 % 값을 인식 못함.
    • 넓이와 높이를 같은 vw으로 주면 넓이가 세로 스크롤의 값때문에 화면에서 넘치게 됨. 대략적인 스크롤 값을 구해 calc값으로 빼주었음.
    • "AOS js 적용"
    • AOS js로 컨텐츠에 zoom-in 효과를 줬더니 가로 스크롤이 생김.
    • 컨텐츠를 div로 감싸 overflow: hidden을 줌.
    • "API 사용 미숙"
    • 책 검색하면 검색결과 나타내기
    • 검색 결과가 기존의 컨텐츠에 영향을 줄 수 없게 검색을 fixed값과 overflow-y:scroll 값으로 레이어와 높이를 따로 주고 기존의 컴텐츠는 display:none으로 감춤.
    • API으로 원하는 위치에 데이터 넣기
    • 위치값이 정확하지 않으면 다른 요소에 데이터가 들어가, 계발자도구를 보며 위치값을 확인.
    • "Json 사용 미숙"
    • Json 데이터를 고려하지 않은 기존css 붕괴
    • 기존 html과 css 값에 일일히 Json 데이터를 적용.
